# SUID-Binary-Explorer
## Install
+ Grant permission to installation script<br>
  <code>$ chmod +x install.sh</code>
+ For installing dependencies and granting permission to other scripts<br>
  <code>$ ./install.sh </code>
+ Launch GUI<br>
  <code>$ ./GUI.py</code>
+ Launch TUI<br>
  <code>$ ./TUI.py</code>
